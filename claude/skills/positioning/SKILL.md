# Positioning (Dunford)

Positioning is deduced, not invented. Work in this order:

1. **Competitive alternatives** - what would customers use if you did not exist (including doing nothing)?
2. **Unique attributes** - what you have that those alternatives do not.
3. **Value** - what those attributes let customers do that they actually care about.
4. **Target segment** - who cares most about that value (be specific; a segment, not "everyone").
5. **Market category** - the frame that makes your value obvious to that segment.
6. **Trend (optional)** - a relevant wave that makes you timely without overreaching.

## Output
A positioning statement, plus a short messaging house: one core message and three supporting pillars, each with a proof point. Specifics over adjectives. Offer to save as a Markdown file openable in Google Docs.
