# [OV] Overview

**What this project is:** Personal dotfiles for macOS (primary) and partial Linux, symlinked into `$HOME`, plus a version-controlled Claude Code "operating layer" under `claude/`.

**Why it exists:** One source of truth for shell, tooling, and Claude Code config that restores on any machine with a single `git clone` plus `./install.sh`.

**Who it's for:** The repo owner (single user, single machine at a time).

**Current status:** shipping (actively used and extended)
